# Twelve Week Year App

## Statements

The Twelve Week year App is an app that was inspired by the book with the same title, The Twelve Week Year by Brian P. Moran. 
The underling premise of this book is to "Get more done in 12 weeks than others do in 12 months." This is my attempt to recreate
the system based off some spreadsheet I came across. 

## Timeline

This project will be worked on during free time. My goal is to track this by hours worked. Ideally I would like to have about 10 hours
per week dedicated.
